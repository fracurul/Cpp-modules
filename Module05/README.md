# Module05 - Exceptions & Advanced Error Management

This module covers exception handling and error management in C++98,
introducing custom exceptions, polymorphism, abstract classes and
design patterns.

### Exercises Overview

- **Ex00 - Bureaucrat**		==> Custom exceptions and validations.
- **Ex01 - Form**			==> Extends exceptions concepts with business logic.
- **Ex02 - AForm**			==> Polymorphism, abstract classes and inheritance.
- **Ex03 - Intern**			==> Factory pattern and form creation.

### Key Concepts

- Custom exception classes.
- Try-catch blocks.
- Abstract base classes (pure virtual functions)
- Polymorphic behavior.
- Factory design pattern.

### Compilation

Each exercise has its own Makefile:
```
cd ex0X && make
```
